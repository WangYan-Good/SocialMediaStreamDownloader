# 📝 项目功能\(Function\)

## douyin

### login
- ⌛更新登录信息

### post

- ⌛根据用户来开启新的子线程下载
- ⌛添加同时下载最大用户数

### live

- ⌛ 添加自动直播下载功能
- ⌛ 添加最大下载数量限制
- ✅添加自定义路径直播下载保存

### APP
- ⌛添加多平台支持
- ⌛支持UI界面下载
- ⌛支持安装可执行文件
- ⌛添加动态命令参数控制下载
- ⌛添加远程下载到指定的服务器位置
- ⌛添加日志功能
